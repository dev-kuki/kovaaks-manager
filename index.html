supabase connection

you need a Supabase project to store your playlists. follow the setup guide below, then paste your credentials here.

how to set up Supabase ↓
  1. go to supabase.com and create a free account
  2. click New project, give it a name (e.g. kovaaks), pick a region close to you, set a database password and save it somewhere
  3. once the project is ready, go to Project Settings → API
  4. copy the Project URL and the anon / public key — paste them below
  5. now go to SQL Editor in the left sidebar and run this to create the two tables:
create table folders (
  id uuid default gen_random_uuid() primary key,
  name text not null,
  created_at timestamptz default now()
);

create table playlists (
  id uuid default gen_random_uuid() primary key,
  name text not null,
  folder_id uuid references folders(id) on delete set null,
  game_tag text,
  notes text,
  file_data jsonb,
  created_at timestamptz default now()
);
  • that's it — paste your URL and key below and hit save
